Коротка інформація про ОС GNU/Linux

GNU/Linux — це вільна та відкрита операційна система, заснована на ядрі Linux, розробленому Лінусом Торвальдсом у 1991 році. Вона є альтернативою закритим операційним системам, як Windows та macOS, і широко використовується на серверах, суперкомп'ютерах, а також у настільних комп'ютерах.

Приклади команд роботи в Linux CLI

  • pwd — відображає поточний робочий каталог.
  • echo — виводить текст або значення змінної.
  • ls — показує вміст каталогу.
  • cd — змінює робочий каталог.
  • touch — створює новий файл.
  • mkdir — створює новий каталог.
  • cp — копіює файли або каталоги.
  • mv — переміщує або перейменовує файли.
  • rm — видаляє файли або каталоги.
  • history — показує історію виконаних команд.

Приклад BASH-скрипта (імітація авторизації)


#!/bin/bash
correct_username="user"
correct_password="password"

echo "Введіть ім'я користувача:"
read username

echo "Введіть пароль:"
read -s password

if [[ "$username" == "$correct_username" && "$password" == "$correct_password" ]]; then
    echo "Авторизація успішна!"
else
    echo "Невірне ім'я користувача або пароль."
fi
